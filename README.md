# 家庭药箱智能管理系统

### 项目简介

**家庭药箱智能管理系统** 是基于 **Multi-Agent 架构** 的 AI 健康管理系统，设计并实现 AI 驱动的家庭药箱管理系统，集成智能检索与健康风险预警，帮助用户安全、高效地管理家庭药品。

**核心价值：**

- 🤖 **Multi-Agent 架构** - Vision Agent + Knowledge Agent + Health Agent 三大AI智能体协同工作
- 🔍 **双模型集成** - 统一封装通义千问（多模态）和 DeepSeek（推理）服务层
- ⚡ **流式输出** - Server-Sent Events (SSE) 实现 AI 思考过程实时展示
- 👨‍👩‍👧 **家庭健康管理** - 集中管理全家人的健康档案和用药情况
- 🔔 **智能提醒** - 定时服药提醒，避免漏服错服
- 🛡️ **安全保障** - AI 自动检查用药风险，保障用药安全

## 快速开始

### 环境要求

- Python 3.10+
- MySQL 5.7+ (或使用 Docker)
- Git

### 1. 克隆项目

```bash
git clone https://github.com/SHChen999/AIManagement.git
```

### 2. 创建虚拟环境

```bash
python -m venv venv

# Windows
venv\Scripts\activate

```

### 3. 安装依赖

```bash
pip install -r requirements.txt
```

### 4. 配置环境变量

编辑 `.env`，填入你自己的 API Key、数据库密码和 SMTP 邮箱配置。

### 5. 启动服务

**终端 1：启动 FastAPI 后端**

```bash
uvicorn main:app --reload  --port 8000
```

**终端 2：启动 Streamlit 前端**

```bash
streamlit run streamlit_app.py
```

### 6. 访问应用

- **前端界面**: http://localhost:8501
- **后端 API 文档**: http://localhost:8000/docs


## 使用说明

### AI 智能体说明

本系统使用多个专业的 AI 智能体协同工作，每个智能体都有独立的思考能力：

| 智能体 | 功能 | AI模型 | 说明 |
|--------|------|--------|------|
| **Vision Agent** | 多模态信息采集智能体 | 通义千问 (Qwen) | 接收用户上传的药盒图片，分析图片信息并保存 |
| **Knowledge Agent** | 知识检索智能体 | DeepSeek | 从药品知识库查询药品详细信息（成分、适应症、禁忌症等） |
| **Health Agent** | 健康管家智能体 | DeepSeek | 分析用药风险，检查过敏史和禁忌症，提供个性化建议 |

### 数据库表说明

| 表名 | 说明 |
|------|------|
| `mlist` | 用户药箱中的药品列表 |
| `family` | 家庭成员健康档案 (AES加密) |
| `medicine` | 药品信息数据库 |
| `reminders` | 用药提醒记录 |

## 数据安全

- 家庭成员健康数据使用 AES 加密存储
- 敏感配置通过环境变量管理
- 邮件密码使用独立授权码，非邮箱登录密码

## 常见问题

**Q: 药品识别失败？**
A: 检查 AI API Key 是否正确配置。

**Q: 邮件提醒没有收到？**
A: 确认 SMTP 配置正确，邮箱已开启 SMTP 服务并获取授权码。
